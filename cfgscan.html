CfgScanner: Next-Generation Memory Forensics & Malware Hunting Tool

Deep dive into volatile memory. Leverage YARA rules and Control Flow Guard (CFG) anomaly detection to hunt down stealthy process injections and Cobalt Strike beacons.

Overview

CfgScanner is a high-performance memory scanner engineered specially for the Windows platform. Unlike traditional antivirus solutions that primarily scan the file system, CfgScanner focuses on the runtime memory space of the active processes. It combines the pattern-matching power of YARA with advanced Windows Control Flow Guard (CFG) bitmap analysis. This dual-engine approach allows it to effectively detect Process Injection, Shellcode exection, and "Fileless" malware hiding in the memory. Additionally, it features bulit-in forensics capabilities for parsing Cobalt Strike configurations directly from infected process memory. Usage Guide

CfgScanner is a command-line interface (CLI) tool designed for flexibility in Incident Response (IR) scenarios.

Bash üìã
# Basic Usage: Scan all processes
CfgScanner.exe -a

# Scan a specific PID with verbose output
CfgScanner.exe -p 1234 -v

# Enable Cobalt Strike config extraction with 10 scanning threads
CfgScanner.exe -a -c -t 10

# Specify a custom output directory
CfgScanner.exe -a -o "C:\Forensics\Reports"

Arguments

Argument Shorthand Description
--all -a Scan all active processes on the system.
--pid -p Target a specific Process ID (PID) for scanning.
--thread -t Set the number of threads for parallel scanning.
--config -c Enable configuration extraction (Triggered on Cobalt Strike detection).
--output -o Specify the directory for log files and memory dumps.
--quiet -q Suppress console output (Silent mode).
